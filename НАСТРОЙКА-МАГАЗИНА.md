# 🛒 Магазин мерча Ростелеком — Инструкция по настройке

## 📋 Что получилось

У вас есть **супер-простой магазин мерча** со следующим функционалом:

✅ **Витрина товаров** — каталог с категориями и поиском  
✅ **Корзина** — добавление товаров, изменение количества  
✅ **Оформление заказа** — форма с контактами  
✅ **Google Таблица** — база данных товаров и заказов  
✅ **Автоматическое списание** — остатки обновляются после заказа  
✅ **Админ-панель** — управление заказами и товарами  
✅ **Уведомления** — можно настроить Telegram

---

## ⚙️ Настройка за 15 минут

### Шаг 1: Создать Google Таблицу (5 мин)

1. Откройте [sheets.google.com](https://sheets.google.com)
2. Создайте новую таблицу → назовите **"Магазин Мерча"**
3. Создайте **3 листа**:

#### Лист 1: `Товары`
Заголовки в строке 1: `id`, `name`, `category`, `price`, `stock`, `image`, `description`

Заполните товарами (пример):
```
1 | Блокнот | basic | 0 | 7 | 📓 | Корпоративный блокнот
2 | Ремувка | basic | 0 | 70 | 🧼 | Ластик для записей
3 | Футболка чёрная | premium | 0 | 62 | 👕 | Качественная футболка
```

#### Лист 2: `Заказы`
Заголовки в строке 1: `id`, `name`, `phone`, `department`, `items`, `total`, `status`, `created_at`, `comment`

(оставьте пустым, заказы будут добавляться автоматически)

#### Лист 3: `Настройки`
```
A1: telegram_bot_token     | B1: (пусто)
A2: telegram_chat_id       | B2: (пусто)
A3: api_key                | B3: rt_merch_secret_key_2026
```

---

### Шаг 2: Настроить Google Apps Script (5 мин)

1. В таблице: **Расширения** → **Apps Script**
2. Удалите весь код, вставьте содержимое файла `мерч-apps-script.gs`
3. Нажмите **💾 Сохранить**
4. Нажмите **▶️ Запустить** → выберите `setUpWebApp`
5. Разрешите доступ

#### Опубликовать веб-приложение:

1. **Разместить** → **Новое развёртывание**
2. Тип: **Веб-приложение**
3. Описание: `Merch API v1`
4. Выполнять от: **Меня**
5. Доступ: **Все** (Anyone)
6. **Развернуть** → скопируйте **URL**

---

### Шаг 3: Вставить URL в HTML (2 мин)

Откройте файл `памятка-мерча.html`, найдите строку:

```javascript
const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/ВАШ_ID_СКРИПТА/exec';
```

Замените на ваш URL из шага 2.

**Повторите** в файле `админка-мерча.html`.

---

### Шаг 4: Проверка работы (3 мин)

1. Откройте `памятка-мерча.html` в браузере
2. Добавьте товары в корзину
3. Оформите заказ
4. Проверьте Google Таблицу — заказ должен появиться
5. Откройте `админка-мерча.html` — заказ должен быть в списке

## 📱 Настройка Telegram-уведомлений (опционально)

### 1. Создать бота

1. В Telegram найдите **@BotFather**
2. Отправьте `/newbot`
3. Введите имя: `Merch Store Bot`
4. Введите username: `rt_merch_bot` (должен быть свободен)
5. Скопируйте **API токен**

### 2. Узнать Chat ID

1. Найдите бота **@userinfobot**
2. Нажмите **Start**
3. Скопируйте **Chat ID**

### 3. Вставить в таблицу

В лист `Настройки`:
- `B1` = токен бота
- `B2` = ваш Chat ID

## 🎨 Как использовать

### Для сотрудников (магазин):

1. Откройте `памятка-мерча.html`
2. Выберите товары → нажмите **"В корзину"**
3. Откройте корзину (иконка 🛒 в шапке)
4. Нажмите **"Оформить заказ"**
5. Заполните форму → **"Подтвердить заказ"**
6. Запомните номер заказа!

### Для администратора:

1. Откройте `админка-мерча.html`
2. Вкладка **Заказы**:
   - Просмотр новых заказов
   - Смена статуса (✓ подтвердить, 📦 готов, ✅ выполнен)
3. Вкладка **Товары**:
   - Редактирование остатков
   - Просмотр наличия
4. Вкладка **Статистика**:
   - Общая сводка по заказам

## 🔧 Технические детали

### Хранение данных:
- **Товары**: Google Таблица (лист `Товары`)
- **Заказы**: Google Таблица (лист `Заказы`)
- **Корзина**: LocalStorage браузера (не теряется при обновлении)

### Обновление остатков:
- Происходит **автоматически** при создании заказа
- Если заказ отменён — нужно вернуть остатки вручную в админке

### Безопасность:
- API ключ для защиты от случайных запросов
- Нет онлайн-оплаты (только пост-оплата или бесплатно)
- Нет авторизации (любой может сделать заказ)

---

## ⚠️ Важные ограничения

1. **Остатки не в реальном времени** — обновляются при загрузке страницы
2. **Нет защиты от "гонок"** — два человека могут заказать последний товар
3. **Корзина локальная** — видна только в одном браузере
4. **Нет email-уведомлений** — только Telegram (опционально)

## 🆘 Если что-то не работает

### Заказы не создаются:
1. Проверьте URL скрипта в HTML
2. Проверьте API ключ в таблице (лист `Настройки`)
3. Откройте консоль браузера (F12) → посмотрите ошибки

### Товары не загружаются:
1. Проверьте названия листов (должны быть точно `Товары`, `Заказы`)
2. Проверьте заголовки столбцов
3. Убедитесь, что Apps Script опубликован как веб-приложение

### Telegram не отправляет:
1. Проверьте токен бота
2. Проверьте Chat ID
3. Убедитесь, что бот не заблокирован
